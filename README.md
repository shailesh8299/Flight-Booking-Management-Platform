# ✈️ SkyBook - Flight Booking Application

A modern, full-stack flight booking web application built with Next.js, TypeScript, and Framer Motion. 

## 🌟 Key Features

- **Next.js App Router**: Optimized for performance and SEO.
- **Mock REST API**: Integrated in-memory backend for demonstration.
- **Real-time Search**: Instant flight search across multiple destinations.
- **Admin Dashboard**: Comprehensive tools for managing flights, users, and bookings.
- **Responsive Design**: Premium UI built with Tailwind CSS and Shadcn/UI.

## 🛠️ Tech Stack

- **Frontend**: Next.js 16 (App Router), React, Framer Motion, Radix UI.
- **Backend**: Next.js API Routes (In-memory storage).
- **Styling**: Tailwind CSS.
- **Icons**: Lucide React.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access Application**:
   Open [http://localhost:3000](http://localhost:3000)

## 🔐 Admin Access

- **Admin Email**: `admin@skybook.com`
- **Password**: `password` (any)

## 📁 Project Structure

- `src/app`: Application routes and API endpoints.
- `src/components`: Reusable UI components.
- `src/hooks`: Custom React hooks for auth and state.
- `src/lib`: Data models, mock database, and utilities.
